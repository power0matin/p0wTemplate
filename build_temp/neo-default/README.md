# Neo Default Theme

The official, extremely clean reference template for Sanaei Panel. 

## Features
- **Clean Layout**: No clutter, just the data your users need.
- **Universal Import**: Built-in Quick Actions for importing to V2RayNG, Shadowrocket, and v2rayN.
- **Dark/Light Mode**: Fully responsive theme switching.
- **Performance**: Zero external dependencies (uses native device fonts and CSS variables).
