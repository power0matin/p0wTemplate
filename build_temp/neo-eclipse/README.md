# Neo Eclipse Theme

A dark, moody, and highly structured template heavily inspired by professional fintech apps. 

## Aesthetics
- **Color Palette**: Deep navy blues, stark whites, and subdued grays.
- **Card-based UI**: Every piece of information is isolated into its own clear, easily digestible card.
- **Data Priority**: Emphasizes Remaining Traffic above all else with a prominent hero section.

## Features
- **Expandable Advanced Info**: Technical details (Up, Down, Total) are hidden inside a clean accordion to prevent clutter.
- **Quick Actions Pill**: Floating button at the bottom for instant configuration imports.
- **QR Support**: Natively renders QR codes cleanly using a lightweight canvas script.
