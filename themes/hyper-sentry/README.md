# HyperSentry Theme

A signature-tier subscription theme for the **p0wTemplate** collection, inspired by the HyperSentry shield-and-lock brand mark.

## Aesthetics
- **Faceted Shield Glass**: Deep-navy glassmorphism with a low-poly facet lattice drifting in the background, echoing the angular shield mark.
- **Sentry Accent System**: A steel-blue → electric-cyan gradient (`--sentry-blue` → `--sentry-cyan`) drives progress bars, buttons, and the pulsing shield avatar.
- **HUD Scan Line**: A subtle animated sweep across the top of the page reinforces the "actively monitored" security feel.
- **Security-Branded Language**: Status badges read *Protected* / *Suspended* instead of generic Active/Disabled, and section headers use shield/lock iconography.

## Features
- **Branded Header**: Shield wordmark with gradient text and an inline SVG shield-and-lock icon.
- **Sentry Avatar**: A pulsing, gradient-filled avatar badge for the client identity card.
- **Dynamic Configs**: Config cards with copy/QR quick actions.
- **Universal Import**: Quick Actions for V2RayNG, Shadowrocket, and v2rayN.
- **Bilingual**: Full English/Persian (RTL) support with the Vazirmatn font.
- **Light/Dark**: Toggleable theme, defaults to system preference.
