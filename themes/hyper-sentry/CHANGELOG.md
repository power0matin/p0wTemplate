# Changelog

All notable changes to the HyperSentry theme will be documented in this file.

## [1.0.0] - 2026-07-06
### Added
- Initial release of HyperSentry — a signature-tier theme for p0wTemplate.
- Faceted shield ambient background with drift animation and HUD scan line.
- Sentry-branded avatar, badges, and status language (Protected/Suspended).
- Gradient accent system (`--sentry-blue` → `--sentry-cyan`) across progress bars and buttons.
- Full bilingual EN/FA (RTL) support.
