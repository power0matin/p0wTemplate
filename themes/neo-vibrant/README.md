# Neo Vibrant Theme

A highly energetic and colorful theme designed to pop off the screen.

## Aesthetics
- **Neon Accents**: Bright, highly saturated colors (cyan, magenta, electric purple) over a deep dark background.
- **Expressive**: Uses bold color choices for progress bars and status indicators.
- **Rounded & Friendly**: Heavy border radiuses and soft shadows.

## Features
- **Expandable Advanced Info**: Technical details are tucked away cleanly.
- **Universal Import**: Built-in Quick Actions floating pill.
- **QR Support**: Native QR rendering.
