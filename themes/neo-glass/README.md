# Neo Glass Theme (iOS 18 Edition)

A stunning, state-of-the-art theme heavily inspired by Apple's iOS 18 design language.

## Aesthetics
- **Frosted Glass (Glassmorphism)**: Intense `backdrop-filter` blur effects over every component.
- **Ambient Lighting**: Vibrant mesh-gradient orbs that glow and bleed through the transparent glass cards.
- **Fluid & Seamless**: Edges blend together to create a unified, ultra-premium look.

## Features
- **Aligned Headers**: Clean, single-line horizontal header structure for absolute minimalism.
- **Dynamic Configs**: A heavily padded, floating list of configurations with beautiful icon actions.
- **Universal Import**: Built-in Quick Actions for importing to V2RayNG, Shadowrocket, and v2rayN.
